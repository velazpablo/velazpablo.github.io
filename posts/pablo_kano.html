Kano activities

In this Kano activities, we discussed a couple of features like bandwidth, networking, and Python. Also, we discussed one specific bash command, better known as tshark. tshark is one of the many network protocol analyzers out there. This tool helps users capture data from live networks or read packets from a previously saved capture file. tshark is a great tool to capture packet loss for the user to verify that there are no issues with a wireless or wired connection.

In this activity, we worked in groups to connect all the peripherals that come with the Kano kit, as well as to configure our user accounts, GUI, and do the first update. After the initial setup was done, we continued to connect the Kano kit to the internet to capture packets from our network use. Such network use is shown in the screenshot below.
